# export-rebranding

Console application to export SQL files.

Example to appsettings:
```
{
    "ConnectionStrings": {
        "Default": "Server=10.10.10.123;User ID=root;Password=123456;Database=test"
    }
}
```

<h3>How to run?</h3>

`dotnet run`

<h3>Technologies</h3>

- .NET 5
- Dapper
- MySQL Connector
